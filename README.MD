*Sample implementations of neural networks that solve the XOR problem from scratch (math libraries only)*

### Compile and run the C++ version:

```
g++ cpp_xor.cpp -o cpp_xor
./cpp_xor
```
### Python version:
```
pip install numpy
python python_xor.py
```
### C version:
```
g++ c_xor.c -o ./c_xor
./c_xor
```
